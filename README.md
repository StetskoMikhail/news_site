# News Site - Strapi Backend + Vue 3 Frontend

Курсовой проект по Back-end разработке на Node.js (4 курс, ПОИТ)

## Технологический стек
- Backend: Strapi v4, Node.js, PostgreSQL
- Frontend: Vue 3, Vite
- Аутентификация: JWT
- Контейнеризация: Docker

## Структура проекта
- \/backend\ - Strapi приложение
- \/frontend\ - Vue 3 приложение
- \docker-compose.yml\ - конфигурация Docker
